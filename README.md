# Machine Learning Aplicado a la Economía

Este repositorio está dedicado al desarrollo e implementación de modelos de **Machine Learning (ML) aplicados al análisis económico y financiero**. Su objetivo es servir como un espacio de experimentación empírica y, al mismo tiempo, como un recurso de aprendizaje estructurado con cuadernos de código abierto.

El repositorio cuenta con cuadernos de Jupyter (`.ipynb`) que abordan distintas técnicas de modelado predictivo y econometría avanzada, organizados de la siguiente manera:

## 📂 Estructura del Repositorio

### 1. Modelos de Regresión
* **`1.ModelosRegresion_BancoMundial.ipynb`**  
  Análisis exploratorio y modelado predictivo utilizando indicadores de desarrollo global. Este cuaderno aborda la limpieza de datos provenientes de la API del Banco Mundial, la detección de multicolinealidad y la evaluación de modelos de regresión avanzados para capturar relaciones económicas complejas.

## 🛠️ Tecnologías y Librerías Utilizadas
* **Lenguaje:** Python 3.x
* **Manipulación y Visualización de Datos:** `pandas`, `numpy`, `matplotlib`, `seaborn`
* **Modelamiento:** `scikit-learn`
* **Extracción de Datos:** `wbgapi` (World Bank API)

## 🎯 Objetivos del Proyecto
1. Aplicar algoritmos de aprendizaje supervisado y no supervisado a problemáticas económicas reales.
2. Explorar dinámicas no lineales en series de tiempo y datos de panel que la econometría tradicional suele omitir.
3. Compartir código limpio, documentado y reproducible para la comunidad académica y de ciencia de datos.
